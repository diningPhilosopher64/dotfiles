If you are in a fresh OS/ new PC. Install i3 and its related stuff, then use the settings files here to  modify accordingly.

polybar, i3blocks, i3-gaps, rofi, dunst, redshift have to installed additionally. 

rofi launch command and mapping is present in autorun.sh

autorun.sh should be placed at ~.
If you are placing it in a different dir, you'll need to modify config file in i3.

install polybar, dunst etc. For polybar you can use the settings in polybar folder.
